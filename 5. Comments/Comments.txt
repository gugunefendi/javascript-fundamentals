Javascript Comments
Comment can be used to explain Javascript code, and can make easy to read.
Comment can be to used for prevent execution of code.

1. Single line comment
Single line comment start with //
Example:
let x = 10; // this is single line comment

2. Multi line comment
Multi line comment start with /* and end with */
Example:
/*
This is a multi line comment
*/

3. Block comment
Block comment is used to comment multiple lines of code.
Example:
/*
let x = 10;
let y = 20;
let z = x + y;
*/
