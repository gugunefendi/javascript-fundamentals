Javascript Syntax:

Javascript Syntax adalah serangkaian aturan tentang bagaimana program Javascript dibangun.

Cara membuat variabel:
var x;
let y;

Cara menggunakan variabel:
x = 10;
y = 'Hello World';


Javascript Values
Syntax Javascript mendefinisikan 2 jenis value:
1. Fixed values / Nilai tetap
2. Variable values / Nilai variabel

Fixed values disebut 'Literals'.
Variable values disebut 'Variabels'.

Javascript Literal
2 aturan dalam membuat fixed values:
    1. 10.50 // decimal value
    2. 1001 // bilangan bulat value
    3. "Hello World" // string value menggunakan kutip dua
    4. 'Hi there' // string value menggunakan kutip satu

Javascript Variable
let x; // x di definiskan sebagai variabel
x = 10; // variabel x diberi nilai 6


Javascript Operators
Javascript menggunakan operator aritmatika ( + - * / ) untuk menghitung nilai
Contoh:
let x = ( 5 + 6 ) * 10;

Javascript menggunakan operator penugasan ( = ) untuk menetapkan nilai ke variabel.
Contoh:
let x, y;
x = 10; // variabel x diberi nilai 10
y = 5; // variabel y diberi nilai 5

Javascript Expression
Expression adalah kombinasi nilai, variabel, dan operator yang menghasilkan nilai baru.
Perhitungan itu disebut evaluasi.
Misalnya, 5 * 10 dievaluasi menjadi 50.
Contoh:
let x = 6 * 6; // variabel x diberi sebuah ekspresi yang menghasilkan nilai baru menjadi 36

Expression juga dapat berisi nilai variabel.
Misalnya x + 5.
Contoh:
let x = 10;
let y = x + 5; // y diberi nilai 15

Expression tidak hanya berupa perhitungan aritmatika namun bisa juga string, boolean, array, object, dll.
contoh:
let x = "Hello " + "World"; // x diberi nilai "Hello World"
let y = true && false; // y diberi nilai false
let z = [1, 2, 3] + [4, 5, 6]; // z diberi nilai [1, 2, 3, 4, 5, 6]

Keyword Javascript
Keyword Javascript digunakan untuk mengidentifikasi tindakan yang akan dilakukan oleh program.
keyword 'let' digunakan untuk mendefinisikan variabel.
Contoh:
let x = 10;

Keyword 'if' digunakan untuk mendefinisikan sebuah kondisi.
Contoh:
if (x > 5) {
  console.log("x is greater than 5");
}

Javascript Comments
// single line comment
/* 
    multi line comment 
*/

Javascript Identifier / Names
Identifier adalah nama yang digunakan untuk mendefinisikan sebuah variabel, fungsi, atau properti.
Aturan penamaan identifier harus diawali dengan:
1. Huruf (AZ - az)
2. Tanda dollar ($)
3. Tanda underscore (_)
Karakter berikutnya dapat berupa huruf, angka, garis bawah atau tanda dollar.

Javascript is Case Sensitive
Semua penamaan identifier di Javascript case sensitive.
Variable lasname dengan lastName adalah dua variabel yang berbeda.
let username = 'Jhon';
let userName = 'Dhoe';

Javascript and Camel Case
Secara historis, programmer telah menggunakan berbagai cara untuk menggabungkan beberapa kata menjadi satu variabel.
1. Hyphens
    first-name, last-name, master-card, inter-city
    // Hyphens tidak di perkenankan di Javascript. Tanda hubung - digunakan untuk pengurangan.
2. Underscore
    first_name, last_name, master_card, inter_city
3. Upper Camel Case (Pascal Case)
    FirstName, LastName, MasterCard, InterCity
4. Lower Camel Case (Camel Case)
    firstName, lastName, masterCard, interCity
    // Programmer Javascript cenderung menggunakan Camel Case yang dimulai dengan huruf kecil.
