Program Javascript

Program komputer adalah daftar "instruksi" yang akan di "eksekusi" oleh komputer.
Dalam bahasa pemrograman, instruksi pemrograman ini disebut pernyataan atau statement.
'Dalam HTML, program Javascript dieksekusi oleh peramban web.'

Javascript Statement.
Statement Javascript terdiri dari:
Value, Operator, Expression, Keyword, dan Comment.
Contoh:
document.getElementById("paragraph").innerHTML = "Hello World"; // ini adalah statement Javascript. Dimana statement ini memberitahu browser untuk menulis Hello World didalam elemen HTML dengan id "paragraph".
Sebagian besar program Javascript mengandung banyak statement Javascript.
Statement-statement tersebut di eksekusi satu per satu, sesuai urutan penulisannya.

Titik koma;
Titik koma (;) memisahkan antara statement yang satu dengan statement yang lain didalam Javascript.
Bila di pisahkan dengan koma, beberapa statement dalam satu baris di perbolehkan.
contoh:
a = 5; b = 6; c = a + b;
Mengahkhiri statment dengan titik koma tidak di wajibkan, tetapi sangat di sarankan.

Whitespace.
Javascript mengabaikan Whitespace atau banyak spasi. Kamu dapat menambahkan spasi untuk memperindah dan memperjelas program kamu.
Contoh salah: let person='Gunawan';
Contoh benar: let person = 'Gunawan';
Praktek yang baik adalah menambahkan spasi di sekitar operator ( = + - * / % ).
Contoh: let x = y + z;

Panjang Baris dan Pemutusan Baris Kode Javascript.
Untuk keterbacaan terbaik, programmer seringkali menghindari baris kode yang lebih panjang dari 80 karakter.
Jika statement tidak muat dalam satu baris, tempat terbaik untuk memutuskannya adalah setelah operator.
Contoh:
document.getElementById("demo").innerHTML =
"Hello Dolly!";

Blok Kode Javascript.
Statement Javascript dapat dikelompokkan bersama dalam blok kode, didalam tanda kurung kurawal {}.
Tujuan blok kode adalah untuk mendefinisikan statemen yang akan di eksekusi bersama.
contoh:
function greet(person) {
    document.getElementById('name').innerHTML = person.name;
    document.getElementById('age').innerHTML = person.age;
}
Kedua statement tersebut akan di eksekusi bersama ketika function greet di panggil.

Keyword Javascript.
Statement Javascript seringkali diawali dengan Keyword untuk mengidentifikasi tindakan Javascript yang akan dilakukan.
Contoh:
var // mendeklarasikan variabel.
let // mendeklarasikan variabel didalam sebuah blok kode.
const // mendeklarasikan variabel yang tidak bisa diubah nilainya.
if // menandai blok statement yang akan di eksekusi pada suatu kondisi.
switch // menandai blok statement yang akan di eksekusi dalam kasus yang berbeda.
for // menandai blok statement yang akan di eksekusi dalam satu loop
function // mendeklarasikan sebuah fungsi.
return // keluar dari suatu fungsi.
try // menerapkan penanganan kesalahan ke blok statement.
