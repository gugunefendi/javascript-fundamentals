Menyisipkan Kode Javascript didalam Halaman HTML.

Gunakan tags <script> {Code Javascript } </script> untuk menyisipkan kode Javascript didalam HTML.
Contoh menyisipkan kode Javascript didalam halaman html:
    <script>
        document.getElementById('paragraph').innerHTML = 'Hello World.';
    </script>

Kamu juga bisa menyisipkan Kode Javascript diantara Tag <head> atau <body>.
Kalau kamu mau menyisipkan kode Javascript di antara tag <head> dan juga tag <body> juga ga masalah, bisa-bisa saja.
